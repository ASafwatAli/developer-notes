# psql tips

## Useful Links

- http://www.postgresqltutorial.com/psql-commands/

## tl;dr

| Command            | Action                          |
| ------------------ | ------------------------------- |
| \c dbname username | Change to database              |
| \l                 | List available databases        |
| \dt                | List tables in current database |
| \d table_name      | Describe table                  |
