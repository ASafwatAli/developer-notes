# Ruby On Rails

## tl;dr

### Installation

`sudo gem install rails (-v 2.3.8)`

Or use `rvm` or `asdf`. Follow those instructions from their respective sites.

### Up and Running

```shell
rails new [project-name]
cd [project-name]
bundle install # may need to `bundle update`
rails server # start the server
```
