# Basic Java Importing

Given the following example to test a `Linked List` Java class, we have the following layout for the `.java` and test files:

```shell
└── src
    ├── main
    │   └── java
    │       ├── LinkedList.java
    │       └── Node.java
    └── test
        └── java
            └── LinkedListTest.java
```
